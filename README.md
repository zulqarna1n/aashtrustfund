Aash’s Trust Fund Website

Aash’s Trust Fund is a charity website dedicated to rescuing, feeding, and sheltering stray cats. It provides information about the trust, displays impact statistics, showcases a gallery of rescued cats, and allows users to donate or volunteer.

Features

Hero Section: Introduction with a call-to-action donate button.

Impact Statistics: Animated counters showing total rescues, donations, expenses, current cats in care, and volunteers.

Mission Section: Details about the trust’s mission, purpose, and activities.

Gallery: Responsive image gallery of rescued cats.

Donate Section: Shows donation methods including Easypaisa, JazzCash, and bank transfer.

Contact Section: Email and WhatsApp contact buttons with professional styling.

Volunteer Form: Users can submit volunteer details via WhatsApp link.

Responsive Design: Works well on desktop and mobile.

Back to Top & Smooth Scroll: Easy navigation between sections.

Technologies Used

HTML5

CSS3 (with gradients and shadows for modern design)

JavaScript (smooth scroll, active navbar highlighting, counter animation)
